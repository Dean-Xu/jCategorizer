jCategorizer is a jquery responsive category manager that will render a given html ul list to an Android looking category manager. It is simple to use and easy to manage.

For documentation, please click here.

Example 1: Simply call jCategorizer() to initialize your category.

$("#jC-container").jCategorizer();

Example 2: Pass "width" and "height" to specify the width and height of your category box when initializing your category.

$("#jC-container2").jCategorizer({ width: '65px', height: '65px' });

Example 3: Pass "iconPerRow" to specify the number of icons displayed per row when initializing your category.

$("#jC-container3").jCategorizer({ width: '65px', height: '65px', iconPerRow: '2' });

Example 4: Pass "overlayType" (shadow/blur) to specify the type of overlay when initializing your category.

$("#jC-container4").jCategorizer({ overlayType: 'blur', width: '65px', height: '65px', iconPerRow: '2' });

Example 5: There are a lot more options you can pass to create your own personalized jCategorizer.

$("#jC-container5").jCategorizer({ overlayType: 'blur', width: '65px', height: '65px', background: '#ffb3b3', fontColor: '#FF0000', border: '2px dotted #FF0000', iconMargin: '3px', iconPerRow: '2' });
